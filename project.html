Τα Project μας

Με θεμέλεια τις γνώσεις μας και εργάλεια την εμπειρία μας έχουμε καταφέρει να δημιουργήσουμε αρκετά projects τα οποία έγιναν και είναι χρήσιμα στην καθημερινότητα μας. Κάποια Projects από αυτά είναι τα εξής:

  1. Τεχνητή Όραση Αναγνώριση ποιο καπάκι είναι :
    Ένα πρόγραμμα στη πλατφόρμα Pycharm σε γλώσσα Python όπου μία κάμερα βλέπαμε το πάνω μέρος από το καπάκι μπουκαλιού νερού και αναγνώριζε τη μάρκα του.

  2. Επίλυση Μαθηματικών Προβλημάτων με την χρήση Cuda :
    Χρησιμοποιώντας τα Threds μίας κάρτας γραφικών Nvidia σε Cuda περιβάλλων επιλύναμε απλά μαθηματικά προβλήματα.

  3. Tic Tac Toe :
    Στη πλατφόρμα Netbeans σε γλώσσα C++ φτιάξαμε το παιχνίδι Tic Tac Toe όπου μπορεί να παίζετε με δύο ή έναν παίχτη και με επιλογή δυσκολίας.

  4. Ναρκαλιευτής :
    Το κλασικό παιχνίδι Ναρκαλιευτής με GUI, δυνατότητα επιλογής στον αριθμό των ναρκών.

Όπως και υπολογίζουμε, έχουμε στο πλάνο μας ακόμα ένα πολύ ενδιαφέρον project στο οποίο θα δώσουμε ιδιαίτερη βάση διότι θα ασχοληθεί αρκετός πληθυσμός με την χρήση του.
